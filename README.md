# 選課系統（Qt/C++）

大一暑期課程項目：基於 Qt Widgets 的 GUI 選課系統，使用 SQLite 存儲用戶與課程數據，實現登錄、選課、退課與課程管理的完整流程，強調數據庫建模與業務規則落地。

## 項目亮點
- Qt Widgets + C++17：桌面端 GUI，交互直觀、流程清晰。
- 輕量數據庫：Qt SQL + SQLite，本地文件即用即跑。
- 選課業務閉環：開放/關閉選課、容量控制、選課與退課。
- 關系數據建模：`userinfo`、`cusinfo`、`enrollment` 等表，包含外鍵與唯一約束。

## 功能概覽
- 登錄與角色：學生 / 教務處 / 輔導員三類入口。
- 學生端：課程列表、已選課程、選課/退課、個人信息查看。
- 管理端：課程新增/刪除、課程信息查看、系統開放/關閉選課。
- 用戶管理：學號注冊、查詢、修改與刪除。

## 技術棧
- 語言：C++17
- GUI：Qt Widgets
- 數據庫：SQLite（Qt SQL 模塊）
- 構建：qmake（`.pro` 工程）

## 目錄結構
- `*.ui`：Qt Designer 界面文件
- `*.cpp/*.h`：業務邏輯與界面交互
- `Tutorial.pro`：Qt 工程配置

## 數據庫設計
應用啟動時會自動創建表（若不存在），默認數據庫文件名為 `hsw.db`：
- `userinfo`：用戶賬號與基礎信息
- `cusinfo`：課程信息（容量與已選人數）
- `enrollment`：選課關系（帶外鍵與唯一約束）
- `system_status`：選課系統開關狀態

> 數據庫文件會生成在運行目錄，建議不要提交到 GitHub。

## 快速開始
1. 使用 Qt Creator 打開 `Tutorial.pro`。
2. 配置 Qt Kit（建議 Qt 5.15+ 或 Qt 6）。
3. 構建並運行，首次啟動會自動初始化數據庫。
